# Warehouse Operations Dashboard — Interview Project

You have 3 hours.

Build an internal dashboard for warehouse operators to manage operational tasks.

This folder contains starter boilerplate only. Implement the missing client and server functionality based on `REQUIREMENTS.md` and `API_REFERENCE.md`.

## Stack

- React + Vite
- React Router
- Tailwind CSS v4
- Axios
- Node.js + Express
- SQL

## Quick start

```bash
cd client
npm install
npm run dev
```

```bash
cd server
npm install
npm run dev
```
